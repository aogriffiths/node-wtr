### Lazy Push To World
```
cd abc ;
git commit -am"updated xyz"; 
git tag -a v0.0.5 -m"tagging"; 
git push origin; 
git push origin --tags;
npm publish;
```

Jumping arround the command prompt:
* Whole line: [ctrl+a] [ctrl+e]
* One word:   [alt+b] [alt+f]
* One letter: [ctrl+b] [ctrl+f]
